# Food Delivery Backend

This repository contains the backend code for the Food Delivery project. It is built with Node.js, 
Express, and MongoDB, and handles user authentication, food menu management, cart operations, and order processing.
The backend exposes RESTful APIs that the frontend consumes. 


The backend is already deployed and accessible at this live link: [https://fooddel-backend-5yfj.onrender.com/].

and the front end link where it is getting used[https://fooddel-frontend-pi.vercel.app/]


To run locally, clone this repository, install dependencies with `npm install`, 
set up your `.env` file with `MONGO_URI`, `JWT_SECRET`, `STRIPE_SECRET_KEY`, and `FRONTEND_URL`, 
then start the server with `npm run server` or `npm start`. Ensure the MongoDB connection URI is valid before running. 



This repository is part of the larger Food Delivery project [https://github.com/Anchalbisht10/Food-delivery],
which also has a separate frontend repository.
The frontend interacts with this backend to fetch and display menu items, manage user carts, and place orders.
